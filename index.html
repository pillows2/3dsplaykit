3DS Playkit

An all new way to play. A huge homebrew starter kit with information, apps, and more. Coming soon.

Apps included:

3DeMore - demo reseter - created by Kyraminol
CHMM2 - custom theme installer - created by Rinnegatamante
CTRMus - mp3/flac player - created by deltabeard
FBI - cia manager/installer - created by Steveice10
FTPD II Turbo - ftp client for 3ds - created by mtheall
JKSM - 3ds game save manager - created by JK
Notepad3DS - notepad for 3ds - created by SomeAmountOfGravy
Notification Bomb - adds random notification - created by TheJosamilu
Playcoin - maximize playcoins at any time - created by MrCheeze
Playkit - information about cfw and the playkit - created by me
Purgification - delete all notifications - created by Rinnegatamante
SCRTool - merge screenshots - created by smealum

Games:

2048 3D - 2048 port to the 3ds - created by MrJPGames
3DSPaint - painting program for the 3ds - created by AlbertoSONIC
Cookie Collector 3D - cookie clicker port to the 3ds - created by Kaisogen
Fidget Spinner 3DS - fidget spinner for 3ds - created by BEPISMAN

Emulation:

PK Emustation - information about emulation in playkit - created by me
Retroarch Nestopia - NES emulator - created by Libretro
Retroarch Snes9x - SNES emulator - created by Libretro
